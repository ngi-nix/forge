# The Problem

The NGI Team is supposed to package 1000+ projects funded by 1600+ NLnet grants.

1. It is impossible to get them packaged by the team members

2. Even if we would package them, it is not possible to maintain them


# The Solution

Scale the load to the upsteam devs and power users of each projects and other
contributors.

## What's needed

The platform for building packages and configurig apps which is:

1. Intuitive and usable for "normal" software devs (no Nix experience, but very
good understanding how to build and run their software)

2. In turn, provides some attractive and unique features for devs and their
users

3. Is built on super powers of Nix and NixOS
